# Football_api
Football live data top leagues with telegram notifications

Key Features
* Real-Time API Integration
* Automated Alert System
* Resource Management


The Logic behind: 
* Shots on Target: `x3` points
* Corner Kicks: `x2` points
* Shots off Target: `x1` point

*If a team's total Pressure Index exceeds the predefined threshold (e.g., 15) while the match score remains tied or close (under 2 total goals), an alert is triggered.*


* Language: Python 3.x
* Libraries: `requests`, `time`
* External APIs: API-Football (v3), Telegram Bot API
* Data Format: JSON
